# DESIGN.md

## 项目定位
**易设计** - 电商人都在用的智能设计平台，专注于电商海报设计、AI商品套图生成、跨境图片处理等商业化功能。

## 气质与意象
- **关键词**: 专业、高效、智能、电商专属
- **场景意象**: 像一位资深电商设计师的智能工作台，工具排列有序，功能触手可及，紫色科技感与橙色活力交织

## 配色方案
| 颜色 | 用途 | 意象来源 |
|------|------|---------|
| 主紫色 `#7C3AED` | 品牌色、按钮、标签 | AI智能感 |
| 渐变紫 `#7C3AED → #EC4899` | 主按钮、卡片标签 | 科技渐变 |
| 橙色 `#F97316` | 推广返现、充值入口 | 电商活力 |
| 中性灰 `#F3F4F6` | 页面背景 | 干净专业 |
| 深灰 `#111827` | 文字、升级会员按钮 | 专业稳重 |

## 字体排版
- **主字体**: Inter（Google Fonts，支持中英文）
- **标题字重**: 600-700
- **正文字重**: 400-500
- **字号层级**: 14px(正文) / 16px(小标题) / 20px(卡片标题) / 24px(页面标题)

## 布局结构
- **左侧导航栏**: 固定宽度 200px，包含首页、AI工具、跨境工具、图片工具、模板、素材、空间
- **顶部横幅**: 粉色渐变通栏广告，右侧立即体验按钮
- **头部功能区**: Logo + 标语 + 推广返现 + 充值 + 升级会员 + 用户头像
- **中央操作区**: 白色圆角卡片，左侧上传按钮，右侧功能推荐

## 组件规范
- **卡片**: 白色背景，圆角 12px，阴影 `shadow-sm`
- **按钮**: 圆角 8px，主按钮紫色渐变，次按钮灰色边框
- **标签**: 小圆角 4px，紫色/橙色背景
- **功能入口**: 200px 宽度卡片，图标+标题+描述

## 交互规范
- **悬停**: 卡片 hover 时轻微上浮 + 阴影加深
- **点击跳转**: 功能卡片点击跳转到海报设计页，URL 带 `?menu=xxx` 参数定位左侧菜单
- **上传交互**: 拖拽上传 + 点击上传，支持图片预览

## 设计禁忌
- ❌ 禁止使用纯蓝色（避免冷冰冰的科技感）
- ❌ 禁止过多圆角（保持专业感，max 12px）
- ❌ 禁止复杂动画（保持效率感，仅轻微 hover 效果）
- ❌ 禁止拥挤布局（保持工具台的清爽感）
